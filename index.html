Character at the nth index


Given a string and a number return the character at the nth index in the given string using javascript

To get the character at the nth index in a given string using JavaScript, you can use the built-in charAt() method.

The charAt() method accepts the character index as an argument and returns its value in the string.

Here is an example that gets the character at the second index of the following string: 

const str = "Hello World";
const char = str.charAt(2);
console.log(char); // Output: "l"

Note that the index of the first character in a string is 0, the second character is 1, and so on. If the index is out of bounds for the string,

charAt() returns an empty string. You can also use square bracket notation [] to get the character at a particular index in a string. If the index is out of bounds for the string, the square bracket notation returns undefined. Here is an example: 

const str = "Hello World";
const char = str[2]; 
console.log(char); // Output: "l"

Therefore, to get the character at the nth index in a given string using JavaScript, you can use either of the following: 

const char = str.charAt(n); 
const char = str[n];

where n is the index of the character you want to get.
